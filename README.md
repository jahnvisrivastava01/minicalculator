# ✨ Modern Calculator

A sleek and responsive calculator built using **HTML, CSS, and JavaScript** with a clean glassmorphism-inspired UI.

---

## 🚀 Features

✅ Perform basic arithmetic operations
✅ Responsive calculator layout
✅ Interactive hover effects
✅ Modern glassmorphism design
✅ Clean and beginner-friendly code structure

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```bash
minicalculator/

├── calculator.html

```

## ⚡ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/minicalculator.git
```

---

### 2️⃣ Open the Project

```bash
cd minicalculator
```

---

### 3️⃣ Run the Project

Simply open:

```bash
calculator.html
```

in your browser.
